### - O que são DOS
~~~
"DOS" pode referir-se tanto a "Disk Operating System", um tipo de sistema operacional usado principalmente em computadores pessoais nas décadas de 1980 e 1990, como o MS-DOS, quanto a "Denial of Service", um ataque cibernético que visa sobrecarregar recursos de uma rede para torná-la indisponível aos usuários.
~~~
